# BigHippo API Coupon

Allows you to retrieve the info of Coupon.

```js
magento.bighippoCoupon.info({
    salesRuleId: Integer
}, callback);
```

Allows you to retrieve the list of Coupon.

```js
magento.bighippoCoupon.list({}, callback);
```

Allows you to delete Coupon.

```js
magento.bighippoCoupon.del({
    salesRuleId: Integer
}, callback);
```
